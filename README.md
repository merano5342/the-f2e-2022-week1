## 本地運行方式
1. 下載專案  
`npm install`

2. 開啟本地 serve  
`npm run start`

3. 在網址輸入，即可觀看  
http://localhost:3000/



## 使用技術
- React
- scss 
- tailwindcss

## 使用版本
- React ^18.2.0
- sass ^1.53.0
- tailwindcss ^3.1.6
